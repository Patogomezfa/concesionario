BUSCADOR REALIZADO CON PHP, MYSQL Y AJAX.

1- Copiar carpeta Proyecto en la carpeta xampp > htdocs (por defecto la direccion es C:\xampp\htdocs)

2- Abrir XAMPP y encender Apache y MySQL

3- En XAMPP abrir el Admin de MySQL

4- Crear Base de Datos con el nombre "autos"

5- En la pestaña Importar seleccionar el archivo autos.sql y hacer click en Continuar

6- Abrir el navegador Web y colocar la url "http://localhost/Proyecto"

******************************

En el buscador principal se podra encontrar los vehículos disponibles en la base de datos, ya sea por
marca, modelo, version y color.
En la seccion administrador se podran cargar los vehiculos nuevos.
